# webapi repository
template repository for deep learning


# setup

## build container

```
make
```

## build vue environment

### create project

```
make vue-init
```

### install libraries for project

```
make vue-install
```

### restore vue ci environment

you should use this task after `git clone` repository

```
make vue-ci
```

# use

# dev environment

```
make vue-dev
```

# prod environment

```
make vue-build
```
